# Aruna - Makeup Artist Portfolio

A premium, professionally designed portfolio website for Aruna, a talented makeup artist specializing in bridal, engagement, and editorial makeup services.

## Features

- **Modern Design**: Clean, elegant interface with luxury branding
- **Responsive**: Fully responsive design that works seamlessly on all devices
- **Portfolio Showcase**: Beautiful gallery of previous work
- **Easy Booking**: Integrated WhatsApp booking system
- **Service Coverage**: Wide service area across South India
- **Client Testimonials**: Real feedback from satisfied clients

## Tech Stack

- **Framework**: Next.js 16.2.4
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Language**: TypeScript
- **UI Components**: Radix UI with class-variance-authority

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Production Build

```bash
npm run build
npm start
```
